# Multi-Domain Data Analysis Portfolio

This portfolio contains five data analysis projects developed using Python, Pandas, and data visualization libraries.

## Projects

1. Supermarket Sales Analysis (Retail)
   - Sales trends
   - Product performance
   - Customer behavior

2. Student Performance Analysis (Education)
   - Subject performance
   - Gender-based analysis
   - Test preparation impact

3. Weather Data Analysis (Weather)
   - Temperature trends
   - Humidity patterns
   - Climate insights

4. COVID-19 Healthcare Analysis (Healthcare)
   - Global case trends
   - Death and recovery analysis
   - Regional comparisons

5. Stock Market Analysis (Finance)
   - Stock price trends
   - Daily return analysis
   - Moving average indicators

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
